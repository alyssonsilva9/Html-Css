Usando CSS

Usamos CSS em HTML de várias maneiras

CSS Inline

Usamos o atributo: “style” na tag que queremos estilizar, exemplo:

<p style="color: blue;">Eu amo programação</p>

Eu amo programação

CSS Interno

Usamos a tag: “<style> </style>” na seção <head> do documento HTML, exemplo:

<style>
p {color: blue;}
</style>

CSS Externo

Criamos um arquivo .css externo e usamos a tag: “<link rel="stylesheet" href="style.css">” na seção <head> para linkar nosso arquivo .css com o documento HTML
