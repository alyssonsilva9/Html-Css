Atributo class

Em CSS usamos “.” para representar uma class

Usamos class quando queremos identificar vários elementos, exemplo:

<p class="cor1">Teste 1</p>

<p class="cor1">Teste 2</p>

Teste 1

Teste 2

Atributo id

Em CSS usamos “#” para representar uma id

Usamos id quando queremos identificar um único elemento, exemplo:

<p id="tel">(00) 00000-0000</p>

(00) 00000-0000
