Shorthand

Propiedade que consigo dizer várias coisas em uma linha, exemplos:

font-style: ;
font-weight: ;
font-size: ;
font-family: ;

Ordem shorthand font: font-style -> font-weight -> font-size -> font-family

font: italic bold 1em sans-serif;

Shorthand de Margin e Padding

margin-top: ;
margin-right: ;
margin-bottom: ;
margin-left: ;

Ordem shorthand margin: margin-top -> margin-right -> margin-bottom -> margin-left

margin: 1px 1px 1px 1px;

padding-top: ;
padding-right: ;
padding-bottom: ;
padding-left: ;

Ordem shorthand padding: padding-top -> padding-right -> padding-bottom -> padding-left

padding: 1px 5px 1px 10px;

A ordem correta do shorthand é: top, right, bottom, left, repectivamente

Border

border-width: ;
border-style: ;
border-color: ;

Ordem shorthand border: border-width -> border-style -> border-color

border: 1px solid blue;

border-top-left-radius: ;
border-top-right-radius: ;
border-bottom-right-radius: ;
border-bottom-left-radius: ;

Ordem shorthand border-radius: border-top-left-radius: ; -> border-top-right-radius: ; -> border-bottom-right-radius: ; -> border-bottom-left-radius: ;

border-radius: 1em 1em 4em 7em;

A ordem correta do shorthand é: top-left, top-right, bottom-right, bottom-left, respectivamente
